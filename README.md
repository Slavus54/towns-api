## Towns API  

### Description  

Open REST API about european towns and context of them: old cathedrals and fortresses.  

You can find any of 475+ towns with rapidly search of *centum.js* library.    

### Technologies  

Server was built on *ExpressJS* with middlewares.      

### URLs  

*https://towns-api.vercel.app* - base URL    

*https://towns-api.vercel.app/towns* - list of towns    

*https://towns-api.vercel.app/town/:title* - sliced list of towns   

*https://towns-api.vercel.app/towns-fragment/:start&:end* - find of town by title     

*https://towns-api.vercel.app/timezones* - list of european towns's timezones   

*https://towns-api.vercel.app/capitals* - list of european capitals with additional information    

*https://towns-api.vercel.app/cathedrals* - oldest european cathedrals of many towns with interesting information

*https://towns-api.vercel.app/fortresses* - fortresses with additional information
